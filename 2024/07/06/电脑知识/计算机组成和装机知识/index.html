计算机组成和装机知识

0.计算机组成

计算机组成包括控制器、运算器、存储器、输入设备、输出设备。

当然这是比较官方学术的说法，不够感官化。

白话一点就是：一个电脑分为硬件系统和软件系统两个组成部分。

硬件系统：CPU、主板、存储器、机箱、电源、输入输出设备……

软件系统：系统软件、应用软件

一般电脑装机就是CPU（中央处理器）、GPU（显卡）、内存、主板、电源、显示器比较重要。

然后是与之搭配的硬盘、散热、机箱、线材、鼠标键盘等等……

最花钱的就是CPU和GPU。

1.中央处理器（CPU）

CPU=控制器+运算器。通俗点说法就是计算机大脑，外形就是个芯片。

CPU最常见的两个厂家Intel和AMD。

截止到今天（20240706）最常见的两个系列是：Intel 的酷睿 CORE 系列和 AMD 的锐龙 RYZEN 系列。

CPU图示：

1.1 Intel 的 CPU

大致可分为 i3、i5、i7、i9 这四大型号，这四个型号中又分了许多的子型号，比如 i7 就有 12700F, 12700K, 12700KF 等。

注意事项：

  1. 同一代的i7基本上可以压着i5打，但是不同代的无法比较，例如12代的i5-12600K就强于11代的i9-11900K。

  2. 散片和盒装的区别：散片是直接提供给整机厂商的OEM版本，没包装，不带散热器。CPU本身和盒装CPU是一样的，散片没有质保，但是CPU故障率极低，还是可以买的。

  3. 散片不是二手CPU，盒装二手还是盒装。散片由于是代加工（OEM）版本，京东天猫买不到，只有淘宝店有。

  4. 想找当前的CPU可以搜CPU天梯图。时间限定在最近一个月即可。

1.2 AMD的CPU

AMD 同样分为 R3、R5、R7、R9 这几个型号。

注意事项：

  1. intel单核频率会高一些，amd的多核处理会强一些。
  2. intel和amd的代数不能对比，厂家不同。可以看CPU天梯图

2.显卡（GPU）

2.1 显卡两大阵营

显卡属于图形处理器。显卡分为集成显卡和独立显卡。集成显卡在中央处理器芯片中，也就是和CPU做到一起。独立显卡就是独立的，插在主板的PCIE扩展槽上。

显卡分为N卡和A卡。也就是 NVDIA（英伟达）和 AMD 两大阵营。生产力的话例如AIGC还是N卡优势大。

显卡图示：

2.2 N卡的简单介绍

N 卡作为现在的主流显卡，是最为人们所推崇的，也是装机比例里面占比最多的显卡品牌，N 卡可以分为一下几个大的类别。

  • Tegra —— 手机和嵌入式设备用的，了解即可
  • GeForce —— 常规民用的显卡系列，打游戏工作都可兼顾，3060,3070等
  • Quadro —— 专业绘图卡，常常配合专业软件使用，RTX6000，RTX8000等
  • Tesla —— 大规模计算，比如深度学习训练，A100等

注意事项：

  1. 查看显卡天梯图
  2. 购买渠道一般就是京东自营、淘宝、拼多多等还有线下实体店
  3. 价格：京东自营≈天猫旗舰店＞实体店＞淘宝第三方≈拼多多
  4. 售后：京东自营＞天猫旗舰店≈实体店≈淘宝第三方≈拼多多
  5. 支持个人送保的品牌，其实在哪里入手都是可以的，售后方面用户可以和厂商直接对接。不支持个人送保的品牌，都建议是优先京东自营。
2.3显卡的品牌及版本号

公版显卡：Nvidia 和 AMD 根据规范生产的显卡。

非公版显卡：第三方品牌拿到Nvidia 和 AMD 的芯片和设计显卡，如七彩虹、蓝宝石、华硕……

N卡阵营：

一线品牌：华硕、微星、七彩虹（准一线）

二三线品牌：索泰、影驰、映众、耕升、翔升、铭瑄、昂达、万丽、必恩威等

A卡阵营：

一线品牌：蓝宝石、华硕

二三线品牌：微星、讯景、撼讯、华擎、盈通、速驹、瀚铠等

注意事项：

  1. 同型号丐版、中端、次旗舰、旗舰的性能差距很小。主要差距在散热、外观上。
  2. 丐版是性价比最高的选择。
  3. 中低端显卡那个便宜买那个，没必要买旗舰型号，好看也看不见。
  4. 现在时间是20240706，再次购买时需要查显卡天梯图。

3.硬盘（HDD和SSD）

3.1硬盘类型

硬盘属于存储器。最重要的就是容量和读写速率。

  1. 容量：B＜KB＜MB＜GB＜TB＜PB。B是byte，也就是最小的存储单位。1KB=1024B，其他同理。
  2. 读写速率：分为读取速率和写入速率，这是两个操作指标，速率并不统一。
  3. 硬盘类型：机械硬盘HDD和固态硬盘SSD。
  4. 机械硬盘：就是磁碟+磁头，容量大，便宜。
  5. 固态硬盘：固态电子存储芯片阵列，读取块，贵。长时间断电可能会丢数据。
  6. 硬盘的接口和协议也会影响传输速率。

3.2硬盘购买
  1. 固态硬盘：速度快、无噪音、贵、长时间断电可能会丢数据。
  2. 机械硬盘：价格便宜、容量大、安全性高，数据易恢复。
  3. 机械硬盘品牌主要有三个：西数、希捷、东芝。
  4. 挑选机械硬盘其实最主要的就是区分垂直盘（CMR）和叠瓦盘(SMR)
  5. 叠瓦盘就需要大的缓存。因此缓存也能简单的判断垂直盘和叠瓦盘，一个2、3T的硬盘配备256MB的缓存，那大概率就是叠瓦盘了。
  6. 游戏盘、视频剪辑盘、组NAS等需要频繁读写的就不能用叠瓦盘了
  7. 固态硬盘分嵌入式（焊死在主板上）、可拔插的sata接口硬盘、M.2接口硬盘。
