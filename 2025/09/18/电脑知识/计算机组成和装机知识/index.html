计算机组成和装机知识

0.计算机组成

计算机组成包括控制器、运算器、存储器、输入设备、输出设备。

当然这是比较官方学术的说法，不够感官化。

白话一点就是：一个电脑分为硬件系统和软件系统两个组成部分。

**硬件系统：**CPU、主板、存储器、机箱、电源、输入输出设备……

**软件系统：**系统软件、应用软件

一般电脑装机就是CPU（中央处理器）、GPU（显卡）、内存、主板、电源、显示器比较重要。

然后是与之搭配的硬盘、散热、机箱、线材、鼠标键盘等等……

最花钱的就是CPU和GPU。

1.中央处理器（CPU）

CPU=控制器+运算器。通俗点说法就是计算机大脑，外形就是个芯片。

CPU最常见的两个厂家Intel和AMD。

截止到今天（20240706）最常见的两个系列是：Intel 的酷睿 CORE 系列和 AMD 的锐龙 RYZEN 系列。

CPU图示：

1.1 Intel 的 CPU

大致可分为 i3、i5、i7、i9 这四大型号，这四个型号中又分了许多的子型号，比如 i7 就有 12700F, 12700K, 12700KF 等。

注意事项：

  1. 同一代的i7基本上可以压着i5打，但是不同代的无法比较，例如12代的i5-12600K就强于11代的i9-11900K。

  2. 散片和盒装的区别：散片是直接提供给整机厂商的OEM版本，没包装，不带散热器。CPU本身和盒装CPU是一样的，散片没有质保，但是CPU故障率极低，还是可以买的。

  3. 散片不是二手CPU，盒装二手还是盒装。散片由于是代加工（OEM）版本，京东天猫买不到，只有淘宝店有。

  4. 想找当前的CPU可以搜CPU天梯图。时间限定在最近一个月即可。

1.2 AMD的CPU

AMD 同样分为 R3、R5、R7、R9 这几个型号。

注意事项：

  1. intel单核频率会高一些，amd的多核处理会强一些。
  2. intel和amd的代数不能对比，厂家不同。可以看CPU天梯图

2.显卡（GPU）

2.1 显卡两大阵营

显卡属于图形处理器。显卡分为集成显卡和独立显卡。集成显卡在中央处理器芯片中，也就是和CPU做到一起。独立显卡就是独立的，插在主板的PCIE扩展槽上。

显卡分为N卡和A卡。也就是 NVDIA（英伟达）和 AMD 两大阵营。生产力的话例如AIGC还是N卡优势大。

显卡图示：

2.2 N卡的简单介绍

**N 卡作为现在的主流显卡，**是最为人们所推崇的，也是装机比例里面占比最多的显卡品牌，N 卡可以分为一下几个大的类别。

  • Tegra —— 手机和嵌入式设备用的，了解即可
  • GeForce —— 常规民用的显卡系列，打游戏工作都可兼顾，3060,3070等
  • Quadro —— 专业绘图卡，常常配合专业软件使用，RTX6000，RTX8000等
  • Tesla —— 大规模计算，比如深度学习训练，A100等

注意事项：

  1. 查看显卡天梯图
  2. 购买渠道一般就是京东自营、淘宝、拼多多等还有线下实体店
  3. 价格：京东自营≈天猫旗舰店＞实体店＞淘宝第三方≈拼多多
  4. 售后：京东自营＞天猫旗舰店≈实体店≈淘宝第三方≈拼多多
  5. 支持个人送保的品牌，其实在哪里入手都是可以的，售后方面用户可以和厂商直接对接。不支持个人送保的品牌，都建议是优先京东自营。
2.3显卡的品牌及版本号

公版显卡：Nvidia 和 AMD 根据规范生产的显卡。

非公版显卡：第三方品牌拿到Nvidia 和 AMD 的芯片和设计显卡，如七彩虹、蓝宝石、华硕……

N卡阵营：

一线品牌：华硕、微星、七彩虹（准一线）

二三线品牌：索泰、影驰、映众、耕升、翔升、铭瑄、昂达、万丽、必恩威等

A卡阵营：

一线品牌：蓝宝石、华硕

二三线品牌：微星、讯景、撼讯、华擎、盈通、速驹、瀚铠等

注意事项：

  1. 同型号丐版、中端、次旗舰、旗舰的性能差距很小。主要差距在散热、外观上。
  2. 丐版是性价比最高的选择。
  3. 中低端显卡那个便宜买那个，没必要买旗舰型号，好看也看不见。
  4. 现在时间是20240706，再次购买时需要查显卡天梯图。

3.硬盘（HDD和SSD）

3.1硬盘类型

硬盘属于存储器。最重要的就是容量和读写速率。

  1. 容量：B＜KB＜MB＜GB＜TB＜PB。B是byte，也就是最小的存储单位。1KB=1024B，其他同理。
  2. 读写速率：分为读取速率和写入速率，这是两个操作指标，速率并不统一。
  3. 硬盘类型：机械硬盘HDD和固态硬盘SSD。
  4. 机械硬盘：就是磁碟+磁头，容量大，便宜。
  5. 固态硬盘：固态电子存储芯片阵列，读取块，贵。长时间断电可能会丢数据。
  6. 硬盘的接口和协议也会影响传输速率。

3.2硬盘购买
  1. 固态硬盘：速度快、无噪音、贵、长时间断电可能会丢数据。
  2. 机械硬盘：价格便宜、容量大、安全性高，数据易恢复。
  3. 机械硬盘品牌主要有三个：西数、希捷、东芝。
  4. 挑选机械硬盘其实最主要的就是区分垂直盘（CMR）和叠瓦盘(SMR)
  5. 叠瓦盘就需要大的缓存。因此缓存也能简单的判断垂直盘和叠瓦盘，一个2、3T的硬盘配备256MB的缓存，那大概率就是叠瓦盘了。
  6. 游戏盘、视频剪辑盘、组NAS等需要频繁读写的就不能用叠瓦盘了
  7. 固态硬盘分嵌入式（焊死在主板上）、可拔插的sata接口硬盘、M.2接口硬盘。

4.内存（memory）

4.1 内存简介

内存属于存储器，是目前计算机用来存储当前运行程序和正在使用的数据的临时性存储设备。内存比从硬盘中读取速度快。但是但它的主要缺点是保存的数据是临时性的，需要供电才能维持，当断电核重启后即被清空需要重新进行写入。

影响内存的参数主要有3个，容量、类型、频率。

名称 说明
容量 内存可以保存的临时数据容量，单位GB。
类型 内存模块的标准技术规格，包含 DDR3、DDR4、DDR5 等，会随着技术的发展持续迭代。
频率 表示内存的传输速率，以 MHz 为单位，通常内存介绍中标注的 2666、3200、4000 等指的就是内存的频率大小。频率越高越流畅

注意点：

  1. 相同的内存条是可以叠加的，但是不能使用不同的类型。例如两个8G组成16G
  2. 叠加过程不能使用不同的类型，且频率不会叠加，如果使用了不同频率的内存，则整体以最低的频率为准。
  3. 两个8G比单一个16G效果好，因为是双通道。
  4. 内存的频率越高，运行大型软件，多任务处理，高负载游戏越流畅。但是还受处理器（CPU和GPU）和主板的限制。有一些高频率内存主板不支持。
  5. 内存都会标注频率，2666MHz、3200MHz、3600MHz、4800MHz、5600MHz、6000MHz等，根据需求选。（注：购买高频内存需要进bios开启xmp，不然D4默频2666MHz运行，D5默认4800MHz运行）
4.2 购买内存（202407）

品牌推荐

  • 宏碁掠夺者、芝奇、威刚、金士顿、海盗船、英睿达、科赋、十铨、光威、阿斯加特、宇瞻等（排名不分先后，只要听过名字的品牌基本都能买）

  • 想支持国产的话不看品牌，认准长鑫颗粒。

是否超频

  • 超频党是选内存颗粒，不超频就是直接选频率。
  • 内存超频，主要是看内存的颗粒，颗粒决定了时序和频率，其实颗粒也决定内存的上限，所以内存品牌其实没颗粒重要。
  • 可以搜DDR4、DDR5内存颗粒天梯图

5.主板（ MainBoard)

5.1主板简介

主板 （MainBoard），是一个用于将 CPU、内存、硬盘、显卡这些独立硬件整合起来并实现正常运转的电路板，是让不同硬件发挥作用的基础平台。

主板=I/O总线+输入输出系统（I/O设备：键盘、鼠标、扫描仪、显示器、数字化仪，读卡机、纸带等。）

主要特征：主板的主要特征就是内部接口和外部接口。

  • 内部接口： CPU 内存、Sata、M.2、PCI-E、电源等
  • 外部接口：USB、网口、音频、集显视频等

主板注意：

  1. 主板对电脑的性能影响相对小，好主板与差主板的区别主要体现在用料做工、外观设计等
  2. 主板可以根据选好的 CPU 来选择，需要注意是 AMD 以及 Intel 的接口是不一样的
  3. 大板接口更多，扩张性好，尺寸大；小板尺寸小，接口也少
  4. 主板供电（供电能满足CPU需求就不会对电脑性能有影响）
  5. 主板不提供性能，花太多钱不划算。

5.2主板购买

主板的高中低端，主要由以下几个方面决定：

  • 接口数量和协议：越高级的主板包含的接口数量越多，协议类型也越先进
  • 制造的工艺：制作的工艺和用料都更好，有更长的寿命
  • 电源供应设计：可以提供更稳定的使用体验和超频的能力
  • 散热解决方案：可以降低元件温度提供更可靠的稳定性
  • 光效 RGB 控制：实现自定义的主板灯源、硬件灯源效果控制
  • 独立芯片组：更好的音频、网络、蓝牙芯片等

主板的主流尺寸：

  • ATX：305mm × 244mm（12 × 9.6 英寸），最常见的主板类型，尺寸较接口较多兼容性最好。
  • MATX：244mm × 244mm（9.6 × 9.6 英寸），尺寸较小，适用于小型机箱，通常具备较好的兼容性。
  • Mini-ITX：170mm × 170mm（6.7 × 6.7 英寸），主板非常小巧，适用于紧凑型机箱，兼容性较差。

购买主要考虑什么

  • 主板供电（供电能满足CPU需求就不会对电脑性能有影响）
  • 拓展性（多内存/显卡插槽、硬盘插槽、IO接口等）
  • WIFI/蓝牙功能（需要这两个功能主板需要选择带WIFI后缀的版本）
  • 双内存槽主板的内存超频能力强于四内存插槽主板，目前最强的（202407）超频主板Z790 APEX就是双槽主板。

主板品牌

  • 主板品牌以御三家为首：华硕、微星、技嘉
  • 技嘉由于我没买就不骂了。
  • 二线的有：映泰、昂达、七彩虹、铭瑄、影驰

6.电源

6.1电源简介

为机箱中所有设备供电。电源主要就是额定功率、电压、80pilus、模块化

名词 说明
额定功率 1. 购买电源时商家所标注的额定功率，如 550W、650W、750W 等，指的是电源的最高输出功率，也就是去除电力转换损耗后，还能为设备输出最高这么大的功率。
2. 选择我们可以直接参考华硕官方给出的电源功率参考建议，华硕官方给出的参考功率是由很大冗余的。
电压 电脑内的各个部件通常工作在不同的电压下，电源需要将 220V 交流电转换为+3.3V、+5V、+12V 的直流电，所以电源所标明的电压就是它可以转换为的电压。
80PLUS 电源上有很多线材，有线材就有电阻，有电阻就有电损，有了电损就会有转换率，选择电源时我们会看到很多会标80Plus铜牌、80Plus金牌之类，这就是电源转换率的标准。一般选择80PLUS铜牌认证以上的电源就可以了，额定功率是指除去电损过之后的功率，不然就是虚标。
模块化 非模组、半模组、全模组，指的是电源输出的线是否可拆卸
非模组电源背部的连接线全部不可拆卸，用不到的只能卷在机箱内；
全模组则是全部可拆卸；
半模组是只有 CPU、主板、GPU 这些必要部件的电源线不可拆卸，其余可拆卸。

注意点：

  1. 不管转换率如何都不会影响额定功率，一款 500W 的电源，不管是铜牌还是金牌，输出都是 500W。转换率影响的是输入功率，转换率低的电源需要从插座取更多的电才能提供 500W 输出，这种影响需要经年累月的使用才会在电费上体现出一定的差异。
  2. 高转换率的电源优势：发热量小，电源静音。省电。
  3. 额定功率大小不代表电源的品质。
  4. 电源转换率不代表电源的品质。
  5. 全模组、非模组不代表电源的品质。设计推荐全模组或半模组，不然强迫症犯了还得买。
  6. ATX 尺寸：标准的桌面计算机机箱电源尺寸
  7. SFX-L 尺寸：应用于较小的紧凑型机箱尺寸
  8. SFX 尺寸：是最小的尺寸，主要用于 ITX 或以下的机箱

6.2电源购买

首选：海韵、安钛克、振华、全汉、台达、EVGA、酷冷至尊、长城

次选：航嘉、鑫谷、先马

品牌不是绝对的选择标准，每个品牌都有高中低端定位的产品，所以具体还要看型号。

7.散热

7.1散热简介

散热很重要，CPU、GPU 都是发热量巨大的元器件，其核心温度高者可达 90°C。大价钱来的不散热会烧坏的。

名词 说明
CPU 散热 CPU 散热需要单独购买，专门为 CPU 进行散热。
GPU 散热 通常显卡会自带散热组件，包含 1-3 个风扇或水冷
机箱散热 由于机箱是一个密闭的空间，从 CPU、GPU 核心处散出来的热量容易堆积在机箱内形成温腔从而影响持续散热，所以需要将机箱内积压的热量导散到外部。
主板散热 主板的元器件、硬盘等等也会散发一定的热量，但量并不大，通常不需要专门针对主板购买独立的散热器件，主板自带则用，没有自带则不用。
被动散热 仅仅通过导热胶将器件的热量传导至表面的金属片上，依靠热量自身的散发降温
主动散热 除了利用硅胶将核心热量传导至金属散热鳍片之外，还依靠风扇主动引导热空气的流向。

散热最简单的方式就是风扇和水冷。

用风扇就要考虑风道不同朝向的风扇明显会打架。

所以风扇的宏观上需要保证一个原则：下进上出、前进后出。

7.2CPU散热

CPU 的散热器有三种形式：下压散热、塔式散热和水冷散热。

下压散热和塔式散热属于风冷，热量主要由金属和空气传导；

水冷散热则使用了专门的水冷液，CPU 的热量由液体相变传导至鳍片处。

注意点：

  • 通常情况下 i5 或 i7 的用风冷散热器就够了，但 i9 的话风冷就不够了，用一体式水冷更好。

  • 不考虑美观性风扇，感觉没啥用还光污染。

  • 下压散热：常见于较低端散热器，下压散热的风扇朝着机箱侧面吹，由于散热面积不大，所以只会用在功耗比较低的 CPU 上。

  • 塔式散热：鳍片和风扇立起来的风冷散热，风扇能够朝着前后吹，塔式散热散热面积较大，可以压功耗更高一些的 CPU。、

  • 水冷散热：水冷散热的冷排——也就是风扇和鳍片——可以直接安装在机箱上，导流管会将在CPU核心处汽化的水冷液导流至冷排处进行液化。

7.3风扇的购买

**便宜：**ID-COOLING XF-12025-SD-K，这个才20

8.机箱

8.1机箱简介

机箱基本上就是装饰品。考虑省钱鞋盒都能用。主要考虑的就是尺寸、硬盘位置、风扇位、开关、I/O接口等。

机箱需要注意的硬件规格：

  • 散热器限高（风冷）
  • 风扇位（水冷规格）
  • 显卡尺寸（显卡长度）
  • 主板规格（ATX、MATX、ITX主板）
  • 电源规格（ATX、SFX电源）
  • 硬盘位（拓展性）
8.2 机箱购买

能装进去硬件就行，价格几十到几千不等。对性能无帮助，鞋盒都行。

尺寸说明：

机箱类型 优点 缺点
ITX机箱 体积小 硬件局限性大，成本高
